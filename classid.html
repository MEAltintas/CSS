Class ve ID Kullanıyorum

Div id bir sayfada belli bir bütünü biçimlendirir ve bu bütün için sayfada bir defa kullanılmalıdır.

Bu div id bu bütünün ayrıcalığını belirtmelidir.

Örnek olarak sol blok kısmının bütününde, sidebar kısmının bütününde gibi.

Div class ise bir bütün için değilde genel ayarlamalar için kullanılır.

Hello CSS ID

I use ID

id komut, kullanıcı kimliğini (UID) ve grup kimliğini (GID) görüntülemek için kullanılmaktadır.

Söz dizimi ID olarak oluşturulmuştur.

ID niteliği , bir HTML etiketi için benzersiz bir kimlik belirtir.

Hello CSS class

I use class

class özelliği, nesneye bir veya birden fazla (aralarında boşluk bırakarak) sınıf atanmasını sağlar.

Sınıflar genelde stil şablonlarından biçimlendirme getirmek amacıyla kullanılsa da, genel amaçla kullanılır.

Aynı sayfada birden fazla HTML etiketine aynı CSS özelliklerini vermek için kullanılır.
